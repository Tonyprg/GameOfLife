Сборка:
> mkdir build
> cd build
> cmake ..
> make

Запуск:
> ./prg

Программа поддерживает следующие команды:
i - увеличить скорость
d - уменьшить скорость
p - остановить/запустить игру
r - заполнить поле случайным образом
c - очистить поле

Так же с помощью мыши можно установть начальную позицию вручную.

Если при запуске программы не получиться прочитать файл config.txt
(неправильно заполнен или вовсе отсутствует) программа попытается
создать (если отстуствовал config.txt) и перезаписать config.txt
настройками по умолчанию. Так же эти настройки можно будет поменять.
